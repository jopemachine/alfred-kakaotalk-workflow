Kakaotalk workflow for Alfred 4

## 📋 Features

* 채팅방 목록 검색

* 채팅방 열기

## 📌 Prerequisite

* 카카오톡 mac 버전 클라이언트 (Tested on 2.7.8 (903))

##  🔨 Change config

`config.json`의 설정을 바꿔 워크플로우의 동작을 변경시킬 수 있습니다.

* **limitCount**

Type: `number`

검색, 표시할 채팅방의 갯수.

* **subtitle**

Type: `string (enum)`

subtitle로 표시할 정보.

* `lastChatting`: 해당 채팅방의 마지막 채팅 표시

* `lastChattingTime`: 해당 채팅방의 마지막 채팅 시각 표시

* `none`: 표시하지 않음
